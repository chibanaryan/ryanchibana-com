This is the first post on my new site. I built it with [Astro](https://astro.build), which lets me write content in Markdown while shipping minimal JavaScript to the browser.

## Why Astro?

Astro's island architecture means static content stays static. Heavy libraries like D3.js and Three.js only load when their components scroll into view. The result is a fast site that can still do interesting things.
